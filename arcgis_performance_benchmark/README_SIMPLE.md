# ArcGIS Python 性能对比测试工具 - 简化版

## 启动方式

**双击启动（无黑窗口）**
```
start_gui_hidden.vbs
```

## 界面说明

### 简化设计
- **顶部设置区**：数据规模、循环次数、预热次数
- **中间日志区**：大型日志窗口，实时显示所有执行信息
- **底部按钮**：
  - ▶ 开始全自动测试（绿色大按钮）
  - 📄 导出报告（测试完成后可用）
  - 清空日志

### 使用流程

1. **调整设置**（可选）
   - 数据规模：小型/中型/大型
   - 循环次数：默认3次
   - 预热次数：默认1次

2. **点击「开始全自动测试」**
   - 自动执行所有5个步骤
   - 实时显示详细日志
   - 完成后自动提示

3. **导出报告**
   - 点击「导出报告」按钮
   - 选择保存位置

## 日志颜色说明

| 颜色 | 含义 |
|------|------|
| 蓝色 | 章节标题、进度信息 |
| 绿色 | 成功消息 |
| 红色 | 错误消息 |
| 橙色 | 警告、提示 |
| 黑色 | 普通信息 |

## 数据规模说明

| 规模 | 数据量 | 预计时间 | 适用场景 |
|------|--------|----------|----------|
| 小型 | 标准1/10 | 5-10分钟 | 快速测试 |
| 中型 | 标准规模 | 30-60分钟 | 日常测试（推荐） |
| 大型 | 超大规格 | 2-4小时 | 学术论文 |

## 注意事项

1. 测试过程中请勿关闭 ArcGIS 程序
2. 点击「停止测试」可随时中断
3. 关闭窗口时会提示确认（如果正在运行）
